# Trusted AI Agents for Business Landing Page

This is a static HTML landing page prepared for deployment on Vercel.

## Deploy with Vercel CLI

1. Install Vercel CLI:
   npm i -g vercel

2. Log in:
   vercel login

3. From this folder, deploy:
   vercel --prod

## Deploy via Vercel Dashboard

1. Upload this folder to a GitHub repository.
2. Go to Vercel > Add New Project.
3. Import the repository.
4. Deploy with default settings.

The homepage is `index.html`.
